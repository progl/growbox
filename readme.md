# Проект "NiceLight GrowBox"

## Описание проекта

Этот проект представляет собой систему гидропоники, предназначенную для автоматического выращивания растений в домашних условиях. Система включает в себя управление освещением, поливом и мониторингом параметров окружающей среды, таких как температура и влажность.

## Функциональные возможности

- Автоматическое управление освещением
- Автоматический полив растений
- Мониторинг температуры и влажности
- Уведомления о состоянии системы

## Требования

- Платформа: PlatformIO
- Микроконтроллер: ESP32
- Датчики: DHT22 (температура и влажность), датчик уровня воды
- Реле для управления освещением и насосом

## Установка и настройка

1. Клонируйте репозиторий проекта:
    ```sh
    git clone https://github.com/yourusername/nicelight_growbox.git
    ```
2. Откройте проект в PlatformIO.
3. Установите необходимые библиотеки:
    ```sh
    pio lib install "DHT sensor library"
    ```
4. Настройте параметры в файле `src/main.cpp` в соответствии с вашими требованиями.
5. Загрузите прошивку на микроконтроллер ESP32.

## Использование

1. Подключите все необходимые датчики и реле к микроконтроллеру ESP32.
2. Включите систему.
3. Мониторьте состояние системы через сериальный монитор или веб-интерфейс (если предусмотрено).
